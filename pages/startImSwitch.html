Start ImSwitch


Click "Start ImSwitch" to launch the ImSwitch application.

Note: ImSwitch must be installed first. If ImSwitch is not installed, you will be prompted to install it. For a more detailed installation process with progress tracking, use the "Install ImSwitch" button from the main menu.

Once started, ImSwitch will run in headless mode and open a web-based interface in this window at http://localhost:8001.
